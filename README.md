Mikamentos es una aplicación de recordatorio de medicamentos diseñada para personas mayores.

PASO 1 - CREAR CATÁLOGO:
Añade tus medicamentos al catálogo con nombre y descripción. Puedes buscar la descripción automáticamente en bases de datos médicas (OpenFDA, CIMA, EMA). El botón 🔄 traduce todas las descripciones al idioma seleccionado.

PASO 2 - PROGRAMAR HORARIOS:
Para cada medicamento del catálogo, selecciona la hora y los días de la semana. Puedes configurar repeticiones (1-10 veces) con intervalo personalizado entre cada una.

PASO 3 - TOMAR MEDICAMENTOS:
A la hora programada suena la alarma y una voz anuncia: "Es hora de tomar [medicamento]". Si hay varios medicamentos, se anuncian uno por uno con un intervalo de silencio configurable (1-5 segundos). La pantalla se mantiene encendida hasta que el usuario descarte la alarma.

Funcionalidades:
• Dictado por voz — pulsa el micrófono para escribir nombres y descripciones hablando
• Botón de escuchar — pulsa el altavoz para oír cualquier descripción en voz alta
• Alarma que funciona con la app cerrada (servicio en primer plano)
• Voz personalizada con título y nombre del usuario
• Repeticiones configurables por medicamento
• Soporte de 6 idiomas: español, inglés, eslovaco, catalán, francés, italiano
• Búsqueda de descripciones médicas online
• Traducción automática de descripciones
• Formato de hora 24h/12h
• Perfil de usuario con reconocimiento de voz
• Compartir la app con amigos por WhatsApp, Quick Share, etc.

Mikamentos is a medication reminder app for elderly people.

STEP 1 - CREATE CATALOG:
Add your medications to the catalog with name and description. You can search descriptions automatically in medical databases (OpenFDA, CIMA, EMA). The 🔄 button translates all descriptions into the selected language.

STEP 2 - SCHEDULE TIMES:
For each medication in the catalog, select the time and days of the week. You can set repetitions (1-10 times) with a custom interval between them.

STEP 3 - TAKE MEDICATIONS:
At the scheduled time, the alarm sounds and a voice announces: "Time to take [medication]". If there are several medications, they are announced one by one with a configurable silence interval (1-5 seconds). The screen stays on until the user dismisses the alarm.

Features:
• Voice dictation — tap the mic to enter names and descriptions by speaking
• Listen button — tap the speaker icon to hear any description read aloud
• Alarm works with the app closed (foreground service)
• Voice customized with user title and name
• Configurable repetitions per medication
• 6 languages: Spanish, English, Slovak, Catalan, French, Italian
• Search medical descriptions online
• Automatic translation of descriptions
• 24h/12h time format
• User profile with voice recognition
• Share the app with friends via WhatsApp, Quick Share, etc.

• Ricerca descrizioni mediche online
• Traduzione automatica delle descrizioni
• Dettatura vocale — premi il microfono per inserire nomi e descrizioni parlando

• Formato ora 24h/12h
• Profilo utente con riconoscimento vocale
